# mentos iOS v0.1 scaffold

This folder contains a SwiftUI scaffold with:
- Sign in with Apple hooks
- Monzo OAuth callback handling (`mentos://oauth/monzo`)
- APNs token registration plumbing
- Tab UI shells (Home, Insights, Goals, Breakthroughs, Settings)

Open in Xcode after creating a target/project named `mentos` and copy files from this directory.
